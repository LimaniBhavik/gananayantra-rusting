Template:

```rust
pub fn calculate_roi(initial: f64, final_value: f64) -> Result<f64, String> {
    if initial <= 0.0 {
        return Err("Initial investment must be greater than zero".into());
    }

    Ok(((final_value - initial) / initial) * 100.0)
}
```
